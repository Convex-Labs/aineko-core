# FastAPI Node

The FastAPI extra node can be used by adding the following to `pyproject.toml`

:   
    ```yaml title="pyproject.toml" hl_lines="2"
    [tool.poetry.dependencies]
    aineko = {version = "^0.2.6", extras=["fastapi"]}
    ```
